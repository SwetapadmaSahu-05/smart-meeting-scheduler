# Smart Meeting Scheduler

## Overview
Smart Meeting Scheduler is a Java console-based application designed to manage meetings efficiently with scheduling conflict detection and role-based authentication.

## Features
- User Login Authentication
- Add Meetings
- View Meetings
- Update Meetings
- Cancel Meetings
- Search Meetings
- Conflict Detection
- File Storage System

## Technologies Used
- Java
- OOP Concepts
- Collections Framework
- File Handling
- ArrayList
- Exception Handling
